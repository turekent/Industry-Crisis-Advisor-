# 2026 产业避险参谋长 - 前端界面使用说明

## 🎉 新增功能

已为您创建了一个专业的Web前端界面，包含以下功能：

### 1. **实时市场动态区**
- 显示最新的5-10条行业动态和大宗商品价格信息
- 自动滚动刷新
- 点击新闻标题可跳转到原文

### 2. **工具介绍区**
- 清晰展示工具的用途和功能
- 列出支持的六大行业

### 3. **行业风险查询区**
- 下拉选择行业
- 可输入详细需求
- 实时流式返回分析结果

## 🚀 启动服务

### 方式一：直接运行（推荐）

```bash
cd /workspace/projects/src
python main.py -m http -p 5000
```

### 方式二：使用 uvicorn

```bash
cd /workspace/projects/src
uvicorn main:app --host 0.0.0.0 --port 5000 --reload
```

## 📱 访问界面

启动服务后，打开浏览器访问：

```
http://localhost:5000
```

### 功能详解

#### 1. 实时市场动态
- **自动加载**：页面打开时自动获取最新资讯
- **手动刷新**：点击"🔄 刷新最新资讯"按钮更新
- **新闻分类**：显示新闻类型（局势动态、价格指标、行业动态等）
- **来源信息**：显示新闻来源和发布时间
- **原文跳转**：点击标题可在新标签页打开原文

#### 2. 行业风险查询
- **行业选择**：从下拉菜单选择要查询的行业
- **详细需求**：可选，输入具体问题或关注点
- **开始分析**：点击按钮或按 Enter 键提交
- **流式输出**：结果实时流式显示，无需等待

#### 3. 风险分析报告
- **局势红绿灯**：🔴 红色警报 / 🟡 黄色预警 / 🟢 绿色提醒
- **动态解读**：用费曼技巧解读国际局势
- **精细影响**：具体的成本影响数据
- **真金白银建议**：可执行的决策建议

## 🔧 故障排查

### 问题1：新闻无法加载
**解决方案：**
- 检查网络连接
- 点击刷新按钮重试
- 查看浏览器控制台错误信息

### 问题2：分析结果未显示
**解决方案：**
- 确认已选择行业
- 检查后端服务是否正常运行
- 查看后端日志：`tail -f /app/work/logs/bypass/app.log`

### 问题3：页面无法访问
**解决方案：**
- 确认服务已启动
- 检查端口是否被占用：`lsof -i:5000`
- 尝试使用其他端口：`python main.py -m http -p 8080`
