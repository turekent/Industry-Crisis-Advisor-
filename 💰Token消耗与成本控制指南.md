### 方案2：设置刷新频率

**如果需要自动刷新，可以降低频率：**

**修改方法：**
```
1. 找到文件：assets/realtime_index.html
2. 找到这行代码（约第400行）：
   
   }, 5 * 60 * 1000);  // 5分钟
   
3. 改为：
   
   }, 30 * 60 * 1000);  // 30分钟
   // 或
   }, 60 * 60 * 1000);  // 1小时
```

**效果：**
```
- 5分钟刷新：每天288次，约9元/月
- 30分钟刷新：每天48次，约1.5元/月
- 1小时刷新：每天24次，约0.7元/月
```

### 方案4：添加使用限制

**在代码中添加限制：**

```javascript
// 每天最多刷新次数
let dailyRefreshCount = 0;
const MAX_DAILY_REFRESH = 20; // 每天20次

function refreshNews() {
    if (dailyRefreshCount >= MAX_DAILY_REFRESH) {
        alert('今日刷新次数已达上限，请明天再试');
        return;
    }
    dailyRefreshCount++;
    // ... 原有刷新逻辑
}
```

**效果：**
```
- 每天最多20次刷新
- 成本可控：约0.02元/天
- 每月约0.6元
```

## ❓ 常见问题

### Q1: 不使用时会产生费用吗？

```
不会！
- 只有访问页面并触发搜索时才消耗token
- 关闭页面后不会继续消耗
```

### Q2: 自动刷新一直开着会怎样？

```
会持续消耗token：
- 每5分钟刷新一次
- 即使您不在看页面
- 建议不用时关闭页面或取消自动刷新
```

### Q3: 静态版和实时版有什么区别？

```
静态版：
- 数据固定，不更新
- 零成本
- 适合演示

实时版：
- 数据实时更新
- 产生token消耗
- 适合获取最新信息
```

### Q4: 如何知道我已经消耗了多少token？

```
在Coze平台查看：
1. 登录 https://www.coze.cn/
```
