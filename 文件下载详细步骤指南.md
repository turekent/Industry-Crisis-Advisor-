## 📥 下载方法详解

### 方法一：在IDE中直接下载（最简单）⭐⭐⭐⭐⭐

#### 步骤1：找到文件
```
左侧文件树 → 项目根目录 → 找到 wechat_industry_advisor.html
```

#### 步骤2：下载文件
- **Windows系统**：右键点击文件 → 选择"下载"或"Export"
- **Mac系统**：右键点击文件 → 选择"下载"或"Export"
- **如果有下载图标**：直接点击文件旁边的下载图标

#### 步骤3：保存到本地
- 选择保存位置（如桌面）
- 文件会下载为 `wechat_industry_advisor.html`

---

### 方法二：复制文件内容（备用方案）⭐⭐⭐

#### 步骤1：打开文件
- 在文件树中点击 `wechat_industry_advisor.html`
- 文件内容会在右侧编辑器中显示

#### 步骤2：复制全部内容
- **Windows**：按 `Ctrl + A`（全选）→ 按 `Ctrl + C`（复制）
- **Mac**：按 `Cmd + A`（全选）→ 按 `Cmd + C`（复制）

#### 步骤3：保存到本地
1. 在电脑上新建一个文本文件
2. 粘贴内容（Ctrl+V 或 Cmd+V）
3. 保存文件，文件名改为：`industry-advisor.html`
4. **重要**：确保文件扩展名是 `.html` 而不是 `.txt`

## 🚀 部署到公网（微信传播）

### 快速方案：使用 GitHub Pages（免费）

#### 1. 创建 GitHub 账号（如果没有）
访问：https://github.com → 注册

#### 2. 创建新仓库
- 点击 "New repository"
- 仓库名：`industry-advisor`
- 设为 Public（公开）

#### 3. 上传文件
- 点击 "uploading an existing file"
- 拖拽 `wechat_industry_advisor.html` 文件
- 点击 "Commit changes"

#### 4. 启用 GitHub Pages
- 进入仓库 Settings → Pages
- Source 选择：main 分支
- 点击 Save

#### 5. 获取链接
几分钟后，访问：
```
https://你的用户名.github.io/industry-advisor/wechat_industry_advisor.html
```

#### 6. 在微信中分享
直接发送链接给朋友，或分享到朋友圈！
