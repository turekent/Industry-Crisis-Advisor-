# 🚀 Vercel 部署教程（小白专用版）

> **适用人群**：零编程基础的小白用户
> **预计时间**：5-10 分钟
> **完全免费**：Vercel 提供免费托管服务

---

## 📋 准备工作

### ✅ 您需要准备：
1. 一台电脑（Windows/Mac 都可以）
2. 一个邮箱（用于注册账号）
3. `wechat_industry_advisor.html` 文件

---

## 🔍 第一步：打开 Vercel 官网

### 1.1 打开浏览器
- Windows 用户：打开 Edge 或 Chrome 浏览器
- Mac 用户：打开 Safari 或 Chrome 浏览器

### 1.2 访问 Vercel 官网
在浏览器地址栏输入：
```
https://vercel.com
```
然后按 **回车键**

![Vercel官网](https://vercel.com/favicon.ico)

---

## 👤 第二步：注册 Vercel 账号

### 2.1 点击注册
在 Vercel 首页右上角，点击 **"Sign Up"**（注册）按钮

### 2.2 选择注册方式（推荐 GitHub）

**方式一：使用 GitHub 注册（最推荐）**
1. 点击 **"Continue with GitHub"**
2. 如果您有 GitHub 账号，直接登录
3. 如果没有，点击 **"Create an account"** 创建一个

**方式二：使用邮箱注册**
1. 点击 **"Continue with Email"**
2. 输入您的邮箱地址
3. 点击 **"Continue"**
4. 去邮箱查看验证码
5. 输入验证码完成注册

---

## 📁 第三步：准备您的文件

### 3.1 获取 HTML 文件
从项目中找到 `wechat_industry_advisor.html` 文件

### 3.2 重命名文件（重要！）
将文件重命名为：
```
index.html
```

**为什么要重命名？**
- Vercel 默认会打开 `index.html` 文件
- 如果不改名，访问链接会显示空白页

### 3.3 创建文件夹（可选但推荐）
在电脑上创建一个文件夹，例如：
- Windows: `C:\Users\你的用户名\Desktop\my-website`
- Mac: `/Users/你的用户名/Desktop/my-website`

把 `index.html` 文件放进去

---

## 🎯 第四步：部署到 Vercel（关键步骤）

### 方法一：拖拽上传（最简单）⭐

#### 4.1 进入部署页面
1. 登录 Vercel 后，点击右上角的 **"Add New..."** 按钮
2. 选择 **"Project"**（项目）

#### 4.2 选择上传方式
1. 您会看到几个选项
2. 找到 **"Upload"** 或 **"Import Git Repository"** 下方的上传区域
3. 有些用户可能会直接看到拖拽区域

#### 4.3 拖拽文件
1. 打开您的文件夹
2. 将 `index.html` 文件直接拖到浏览器窗口
3. 或者点击 **"Browse"** 选择文件

#### 4.4 等待上传
- 上传过程通常只需要几秒钟
- 您会看到一个进度条

#### 4.5 点击部署
1. 上传完成后，点击 **"Deploy"**（部署）按钮
2. 等待 10-30 秒
3. 您会看到一个庆祝动画 🎉

---

### 方法二：使用 Git 上传（适合想学习的用户）

<details>
<summary>📖 点击展开查看 Git 方式（可选）</summary>

#### 4.1 创建 GitHub 仓库
1. 登录 GitHub.com
2. 点击右上角 **"+"** → **"New repository"**
3. 仓库名称填写：`industry-advisor`
4. 点击 **"Create repository"**

#### 4.2 上传文件到 GitHub
1. 点击 **"uploading an existing file"**
2. 拖拽 `index.html` 文件
3. 点击 **"Commit changes"**

#### 4.3 在 Vercel 导入
1. 回到 Vercel
2. 点击 **"Add New..."** → **"Project"**
3. 选择您的 GitHub 仓库
4. 点击 **"Deploy"**

</details>

---

## 🎉 第五步：获取您的网站链接

### 5.1 部署成功后
您会看到一个庆祝页面，显示：
```
🎉 Congratulations! Your project has been deployed.
```

### 5.2 获取链接
有两种方式获取链接：

**方式一：点击访问**
1. 点击页面上的 **"Visit"** 按钮
2. 您的网站会在新标签页打开
3. 复制浏览器地址栏的链接

**方式二：从 Dashboard 获取**
1. 回到 Vercel 首页
2. 点击您的项目名称
3. 在 **"Domains"** 区域可以看到链接
4. 格式类似：`https://industry-advisor-xxx.vercel.app`

## ❓ 常见问题解决

### 问题1：打开链接显示空白页

**原因**：文件名不是 `index.html`

**解决方法**：
1. 确认文件名是 `index.html`（不是 `index.html.txt`）
2. 重新上传正确的文件

---

### 问题2：样式乱掉了

**原因**：复制文件时格式出错

**解决方法**：
1. 重新下载原始的 `wechat_industry_advisor.html` 文件
2. 重命名为 `index.html`
3. 重新上传

---

### 问题3：链接打不开

**原因**：部署未成功或链接错误

**解决方法**：
1. 回到 Vercel Dashboard
2. 检查项目状态是否显示"Ready"
3. 如果显示"Error"，点击查看错误日志
4. 重新部署

---

### 问题4：想修改内容怎么办？

**方法一：在 Vercel 上修改**
1. 登录 Vercel
2. 点击您的项目
3. 点击 **"Files"** 标签
4. 找到 `index.html`
5. 点击编辑
6. 修改后点击 **"Commit"**

**方法二：重新上传**
1. 在本地修改 `index.html` 文件
2. 回到 Vercel 项目页面
3. 点击 **"Deployments"** 标签
4. 点击最新部署旁的三个点 **"..."**
5. 选择 **"Redeploy"**

---

### 问题5：想要自定义域名

<details>
<summary>📖 点击展开查看域名设置</summary>

1. 登录 Vercel
2. 进入您的项目
3. 点击 **"Settings"** 标签
4. 点击 **"Domains"**
5. 输入您想要的域名
6. 点击 **"Add"**

**注意**：
- 免费的 `.vercel.app` 域名可以直接使用
- 自定义域名（如 `.com`）需要购买

</details>

## 🎨 进阶技巧

### 技巧1：添加网站图标

在 `<head>` 标签中添加：
```html
<link rel="icon" href="https://您的图标链接.ico">
```

### 技巧2：添加网站描述

在 `<head>` 标签中添加：
```html
<meta name="description" content="产业避险参谋 - 针对6大行业提供战争实时动态的成本预警和采购建议">
```

### 技巧3：查看访问统计

1. 登录 Vercel
2. 进入您的项目
3. 点击 **"Analytics"** 标签
4. 启用 Analytics（免费）
