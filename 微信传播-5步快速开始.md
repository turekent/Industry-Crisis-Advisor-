# 🎯 微信传播 - 5步快速开始

## 📍 操作步骤

### ✅ 第1步：复制代码
```
1. 左侧文件树 → 找到 wechat_industry_advisor.html
2. 点击打开
3. Ctrl+A 全选 → Ctrl+C 复制
```

### ✅ 第2步：保存文件
```
1. 电脑上新建文本文件
2. 粘贴代码
3. 保存为：index.html（保存类型选"所有文件"）
```

### ✅ 第3步：打开Vercel
```
浏览器访问：https://vercel.com
点击 "Sign Up" 注册（可用GitHub账号）
```

### ✅ 第4步：上传部署
```
1. 登录后 → "Add New..." → "Project"
2. 选择 "Upload"
3. 拖拽 index.html 文件
4. 点击 "Deploy"
```

### ✅ 第5步：获取链接
```
等待30秒 → 获得链接：https://xxx.vercel.app
在微信中发送这个链接即可！
```
