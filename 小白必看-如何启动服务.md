## ❓ 常见问题

### Q1: 提示"python不是内部或外部命令"

**解决方法：**
```
您需要先安装 Python：
1. 访问 https://www.python.org/downloads/
2. 下载 Python 3.12
3. 安装时勾选"Add Python to PATH"
4. 重启命令行
5. 再次运行启动命令
```

### Q2: 提示"ModuleNotFoundError: No module named 'xxx'"

**解决方法：**
```
安装缺失的依赖：
pip install -r requirements.txt
```

### Q3: 端口被占用

**解决方法：**
```
修改 .env 文件中的端口：
PORT=5001

然后访问：
http://localhost:5001
```

### Q4: 无法访问 localhost

**可能原因：**
```
1. 服务未启动 → 重新运行启动命令
2. 浏览器问题 → 换个浏览器试试
3. 防火墙阻止 → 允许 Python 访问网络
```

## 📊 当前配置说明

**您的 .env 文件内容：**
```env
# 基础配置
PORT=5000
DEBUG=true
LOG_LEVEL=INFO

# AI配置（系统会自动使用环境变量）
COZE_WORKLOAD_IDENTITY_API_KEY=请填入您的API_Key
COZE_INTEGRATION_MODEL_BASE_URL=https://api.coze.cn
```

**⚠️ 注意：**
- 系统会优先使用环境变量中的配置
- 您已经配置好了所有环境变量
- 所以 .env 文件中的"请填入您的API_Key"不会影响使用
- 如果环境变量失效，才需要填写 .env 文件
