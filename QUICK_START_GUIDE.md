## 如果需要重启服务

### 1. 停止当前服务
```bash
# 查找进程ID
lsof -i:5000

# 停止进程（替换PID为实际进程ID）
kill -9 PID
```

### 2. 重新启动
```bash
cd /workspace/projects/src
python main.py -m http -p 5000
```

### 3. 使用其他端口
如果5000端口被占用，可以使用其他端口：
```bash
cd /workspace/projects/src
python main.py -m http -p 8080
```

然后访问：`http://localhost:8080`

## 故障排查

### 问题1：端口被占用
**解决方案**：使用其他端口
```bash
python main.py -m http -p 8080
```

### 问题2：无法访问
**解决方案**：
1. 检查服务是否运行：`curl http://localhost:5000/health`
2. 查看日志：`tail -f /app/work/logs/bypass/app.log`
3. 重启服务

### 问题3：新闻加载失败
**解决方案**：
1. 检查网络连接
2. 查看日志错误信息
3. 手动刷新新闻
