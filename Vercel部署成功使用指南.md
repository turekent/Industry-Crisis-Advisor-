## 📍 一、找到您的网站链接

### 方法1：在 Vercel Dashboard 查看

```
1. 打开浏览器，访问 https://vercel.com
2. 登录您的账号
3. 点击您的项目名称（如：industry-advisor）
4. 在 Overview 页面可以看到：
   
   ┌─────────────────────────────────────────┐
   │  🌐 Preview                             │
   │  https://industry-advisor-xxx.vercel.app │
   │  [Visit] 按钮                           │
   └─────────────────────────────────────────┘
   
5. 点击 "Visit" 按钮打开网站
6. 或者复制这个链接
```

### 方法2：从部署成功页面获取

```
部署成功时，Vercel 会显示：

┌─────────────────────────────────────────┐
│  🎉 Congratulations!                    │
│  Your project has been deployed         │
│                                         │
│  🔗 https://xxx.vercel.app             │
│  [Visit] [Go to Dashboard]              │
└─────────────────────────────────────────┘

直接复制这个链接即可！
```

## 📱 二、在微信中使用

### 步骤1：复制链接
```
1. 在浏览器地址栏点击链接
2. 全选（Ctrl+A 或 长按）
3. 复制（Ctrl+C 或 右键菜单）
```

### 步骤2：在微信发送

**发给朋友/群：**
```
1. 打开微信
2. 选择要发送的聊天窗口
3. 在输入框粘贴链接（Ctrl+V 或 长按粘贴）
4. 发送

示例：
【产业避险参谋】
针对PCB、游艇等6大行业，提供实时成本预警
点击查看最新动态 → https://xxx.vercel.app
```

**发朋友圈：**
```
1. 打开微信 → 发现 → 朋友圈
2. 点击右上角相机图标
3. 可以添加配图（可选）
4. 在文字描述中粘贴链接
5. 点击"发表"

示例：

避免信息差亏损，点击直接用
https://xxx.vercel.app
```

### 步骤3：在微信中打开
```
1. 在微信中点击链接
2. 自动打开微信内置浏览器
3. 可以正常浏览和使用所有功能
```

## 🔧 四、常见问题解决

### Q1: 链接打不开？

**检查步骤：**
```
1. 回到 Vercel Dashboard
2. 查看项目状态是否显示 "Ready"
   ✅ Ready = 部署成功
   ❌ Error = 部署失败
   
3. 如果显示 Error：
   • 点击查看错误日志
   • 检查 index.html 文件是否正确
   • 重新部署
```

### Q2: 页面显示空白？

**可能原因：**
```
❌ 文件名不是 index.html
   → 解决：重命名文件为 index.html，重新上传

❌ 文件内容错误
   → 解决：重新下载原始文件，重新上传
```

### Q3: 样式乱了？

**解决方法：**
```
1. 清除浏览器缓存
2. 刷新页面（Ctrl+F5）
3. 如果还不行，重新下载文件上传
```

### Q4: 微信中打不开？

**解决方法：**
```
1. 用其他浏览器（Safari/Chrome）测试链接
2. 如果其他浏览器能打开，但微信不行：
   • 可能是微信缓存问题
   • 让朋友试试能否打开
   • 或者用"复制链接"方式分享
```

## 📊 五、链接格式说明

### Vercel 默认域名格式：
```
https://[项目名]-[随机字符].vercel.app

示例：
https://industry-advisor-abc123.vercel.app
https://industry-risk-advisor.vercel.app
```

### 如果想自定义域名：
```
1. 在 Vercel 项目中 → Settings → Domains
2. 输入您的域名
3. 按提示配置 DNS
4. 等待生效（通常1-24小时）

注意：
• .vercel.app 域名免费
• 自定义域名（.com等）需要购买
```

## 💡 六、使用技巧

### 技巧1：收藏到手机桌面

**iPhone:**
```
1. 在 Safari 打开链接
2. 点击分享按钮（方框箭头）
3. 选择"添加到主屏幕"
4. 像App一样使用
```

**Android:**
```
1. 在 Chrome 打开链接
2. 点击菜单（三个点）
3. 选择"添加到主屏幕"
4. 像App一样使用
```

### 技巧2：生成二维码

```
1. 访问二维码生成网站（如草料二维码）
2. 粘贴您的链接
3. 生成二维码
4. 保存图片，发朋友圈或打印
```

### 技巧3：查看访问统计

```
1. 登录 Vercel
2. 进入项目 → Analytics
3. 启用 Analytics（免费）
4. 查看访问量、地区等数据
```
