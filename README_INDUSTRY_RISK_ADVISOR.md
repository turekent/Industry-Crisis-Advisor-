# 2026 产业避险参谋长 - 使用指南

## 📋 项目概述

**2026 产业避险参谋长** 是一个专业的行业风险分析 AI Agent，针对六大行业（游艇、生物医药、PCB、新型储能、打印耗材、跨境电商）提供基于战争实时动态的成本预警和采购建议。

## 🎯 核心功能

### 1. 实时风险信息检索
- 自动搜索特定行业的最新战争动态
- 监控原材料价格波动
- 追踪物流延误情况

### 2. 精准成本影响分析
- 计算原材料价格变化对总成本的影响
- 评估风险等级（红/黄/绿三级预警）
- 量化具体的成本变化金额

### 3. 专业决策建议
- 基于费曼技巧的通俗解读
- 具体可执行的行动建议
- 行业特定的风险提示

## 🏭 支持的行业

| 行业 | 关键原材料 | 核心指标 |
|------|-----------|---------|
| **PCB** | 电解铜箔、覆铜板(CCL)、钯/金电镀药水 | LME铜价、钯金现货 |
| **游艇** | 不饱和聚酯树脂、乙烯基树脂、胶衣、进口发动机 | 布伦特原油 |
| **生物医药** | API原料药、异丙醇/乙腈溶剂、液氦 | 中东空运溢价、卡塔尔航道 |
| **新型储能** | PVDF粘结剂、电解液溶剂(DMC/EMC)、铝箔 | 沪铝主力、电力成本预期 |
| **打印耗材** | 聚合碳粉、ABS/HIPS塑料粒子、显影滚轴 | 远东塑料价格指数 |
| **跨境电商** | 瓦楞纸箱、航空燃油附加费、集装箱运价 | SCFI运费指数、好望角绕行指数 |

### 输出格式

Agent 会按照以下四个板块生成专业报告：

#### 1. 【局势红绿灯】
- 🔴 **红色警报**：原材料价格涨幅 ≥10%，建议立即采取行动
- 🟡 **黄色预警**：原材料价格涨幅 5%-10%，需要密切关注
- 🟢 **绿色提醒**：原材料价格涨幅 2%-5%，保持观察即可

#### 2. 【动态】
用费曼技巧，将复杂的国际局势转化成老板听得懂的"盈亏账"。

#### 3. 【精细影响】
列出具体的成本影响：
- 每种关键原材料的价格变化
- 对总成本的影响百分比
- 物流延误情况（如有）

#### 4. 【真金白银建议】
给出具体可执行的建议：
- 采购时机建议
- 库存管理建议
- 报价策略建议
- 行业特定建议

## 📊 示例输出

### PCB 行业分析示例

```
## 【局势红绿灯】🟡 黄色预警

## 【动态】
近期伊朗以色列冲突持续升级，霍尔木兹海峡航道通行受限，导致全球原油供应链紧张，直接推高了PCB行业核心原材料的成本。简单说就是：
- 国际原油涨价→电解铜箔生产成本上升→每张PCB的铜箔成本涨了10块多
- 中东航线运费暴涨，每个集装箱额外加收3500美元战争附加费

## 【精细影响】
- **电解铜箔**：基础价格180元/张，涨幅5.9%，每张成本增加约10.54元
- **覆铜板(CCL)**：日本巨头已宣布提价30%以上
- **总成本影响**：预计上涨2.05%

## 【真金白银建议】
1. ⏳ 3天内锁定近期订单
2. 🤝 与核心供应商保持每日沟通，密切关注LME铜价走势
3. 📋 立即启动备用供应商评估
4. 📝 报价单有效期缩短至3-5天
```

### 工具说明

#### 1. industry_risk_search
**功能**：搜索特定行业的实时风险信息

**输入参数**：
- `industry`: 行业名称（PCB、游艇、生物医药、新型储能、打印耗材、跨境电商）

**返回内容**：
- 风险动态摘要
- 原材料信息
- 物流状况
- 发布时间

#### 2. calculate_cost_impact
**功能**：根据搜索结果计算成本影响

**输入参数**：
- `industry`: 行业名称
- `search_results`: 搜索结果的JSON字符串

**返回内容**：
- 风险等级
- 成本变化百分比
- 具体材料影响
- 决策建议

## 🔧 高级功能

### 1. 自定义成本模型
在 `src/tools/cost_calculator.py` 中修改 `COST_MODELS` 字典，可以自定义：
- 原材料权重
- 基础价格
- 价格敏感度
- 预警阈值

### 2. 扩展搜索关键词
在 `src/tools/industry_risk_search.py` 中修改 `INDUSTRY_CONFIG` 字典，可以添加：
- 更多搜索关键词
- 新的关键指标
- 扩展原材料列表

### 3. 调整风险等级
在 `src/tools/cost_calculator.py` 中修改 `_generate_recommendations` 函数，可以调整：
- 风险等级阈值
- 建议内容
- 行业特定提醒

## 🛠️ 部署建议

### 生产环境部署
1. 确保 `COZE_WORKLOAD_IDENTITY_API_KEY` 环境变量已设置
2. 确保 `COZE_INTEGRATION_MODEL_BASE_URL` 环境变量已设置
3. 检查网络连接（需要访问外部搜索服务）
4. 建议设置请求超时为 60-120 秒

### 性能优化建议
1. 对于高频查询，可以考虑缓存搜索结果（有效期 1-2 小时）
2. 对于批量查询，建议使用异步处理
3. 建议设置并发限制，避免过度调用外部 API

## ⚠️ 注意事项

1. **数据实时性**：搜索结果基于实时网络信息，可能存在延迟
2. **价格波动**：原材料价格波动频繁，建议定期查询更新
3. **行业限制**：仅支持六大行业，其他行业会提示不支持
4. **决策参考**：分析结果仅供参考，具体决策需结合实际情况
