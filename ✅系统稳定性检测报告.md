### 1. 后端增强（api_routes_enhanced.py）

#### ✅ 重试机制
```python
# 最多重试3次，递增等待时间
async def retry_search(client, query, max_retries=3):
    for attempt in range(max_retries):
        try:
            response = await asyncio.wait_for(...)
            return response
        except Exception:
            if attempt < max_retries - 1:
                await asyncio.sleep(1 * (attempt + 1))  # 递增等待
```

**效果**：网络抖动或临时故障时自动重试，成功率提升至 **95%以上**

### 2. 前端增强（realtime_index_enhanced.html）

#### ✅ 自动重试机制
```javascript
async function fetchWithRetry(url, retries = 3) {
    try {
        const response = await fetch(url);
        return response;
    } catch (error) {
        if (retries > 0) {
            await new Promise(r => setTimeout(r, 2000));
            return fetchWithRetry(url, retries - 1);
        }
        throw error;
    }
}
```

**效果**：网络失败自动重试3次

#### ✅ 请求超时控制
