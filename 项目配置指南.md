## 二、项目简介（README）

### 英文版

```markdown
# Industry Risk Advisor

> AI-powered risk analysis tool for 6 major industries, providing real-time cost warnings and procurement recommendations based on geopolitical dynamics.

## 🎯 Overview

Industry Risk Advisor is an intelligent risk analysis system designed for:
- 🏭 **PCB** (Printed Circuit Board)
- 🛥️ **Yacht Manufacturing**
- 💊 **Biopharmaceuticals**
- 🔋 **Energy Storage**
- 🖨️ **Printing Consumables**
- 📦 **Cross-border E-commerce**

## ✨ Key Features

- **Real-time Market Intelligence**: Automatic data collection from web searches
- **Cost Impact Analysis**: Calculate material cost changes with precision
- **Risk Level Assessment**: Traffic light system (Red/Yellow/Green)
- **Actionable Recommendations**: Feynman-style explanations for business owners
- **Multi-source Data**: Professional APIs + Web search + Fallback data

## 🚀 Quick Start

1. Clone the repository
2. Install dependencies: `pip install -r requirements.txt`
3. Configure API keys in `.env`
4. Run: `python src/main.py`

## 📊 Output Format

### 【Risk Traffic Light】🟡 Yellow Warning
### 【Dynamics】
- Real-time geopolitical situation analysis
- Material price fluctuation tracking

### 【Detailed Impact】
- Material-specific cost changes
- Percentage impact analysis

### 【Actionable Recommendations】
- Specific steps for risk mitigation
- Timeline for action

## 🛠️ Tech Stack

- **Backend**: Python, FastAPI, LangChain, LangGraph
- **AI Model**: Doubao (doubao-seed-1-8-251228)
- **Data Source**: Web Search SDK, Professional Price APIs
- **Frontend**: HTML/CSS/JavaScript (WeChat-friendly)

## 📝 License

MIT License

## 🤝 Contributing

Contributions are welcome! Please feel free to submit a Pull Request.
```

## 三、配置文件分析

### 1. ✅ **需要 .gitignore**

**原因**：
- ✅ 排除敏感信息（API keys、环境变量）
- ✅ 排除 Python 缓存文件
- ✅ 排除虚拟环境
- ✅ 排除 IDE 配置

**创建文件**：`.gitignore`

---

### 2. ✅ **需要 LICENSE**

**原因**：
- ✅ 开源项目必须有许可证
- ✅ 明确使用权限
- ✅ 保护作者权益

**推荐**：MIT License（最宽松）

**创建文件**：`LICENSE`

---

### 3. ❓ **是否需要 template？**

**分析**：

| Template 类型 | 是否需要 | 原因 |
|--------------|---------|------|
| GitHub Issue Template | ✅ 推荐 | 规范问题提交 |
| GitHub PR Template | ⚠️ 可选 | 规范代码提交 |
| Project Template | ❌ 不需要 | 不是模板项目 |

**建议**：
- 如果只是部署使用，**不需要**
- 如果想开源让他人使用，**建议添加 Issue Template**
