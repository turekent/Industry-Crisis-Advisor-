# ⚡ 5分钟快速部署指南

> 超级简化版，跟着做就行！

---

## 🎯 三步完成部署

### 第一步：准备文件（1分钟）

```
1. 找到文件：wechat_industry_advisor.html
2. 重命名为：index.html（必须改！）
3. 放在桌面上，方便找
```

---

### 第二步：上传部署（3分钟）

```
1. 打开浏览器，访问：https://vercel.com
2. 点击右上角 "Sign Up"（注册）
3. 选择用 GitHub 或 Email 注册
4. 注册完成后，点击 "Add New..." → "Project"
5. 找到 "Upload" 区域，把 index.html 拖进去
6. 点击 "Deploy"（部署）
7. 等待30秒，看到庆祝动画 🎉
```

---

### 第三步：获取链接（1分钟）

```
1. 部署成功后，点击 "Visit" 按钮
2. 复制浏览器地址栏的链接
3. 链接格式：https://xxx.vercel.app
4. 发到微信即可！
```

## ❌ 常见问题

### Q: 打开是空白页？
**A: 文件名不是 index.html，重新改名上传**

### Q: 样式乱了？
**A: 重新下载原始文件，再上传**

### Q: 链接打不开？
**A: 等待1-2分钟，或重新部署**
