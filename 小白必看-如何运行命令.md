# 🎯 小白必看：如何运行命令

---

## 📍 方法1：Windows系统（最常用）

### 步骤1：打开命令提示符

**方法A：最快的方法**
```
1. 按键盘上的 Win + R 键（同时按）
   Win键就是那个有Windows图标的键 ⌨️
   
2. 会弹出一个小窗口

3. 输入：cmd

4. 按回车键 Enter

5. 命令提示符窗口就打开了！
```

**方法B：搜索打开**
```
1. 点击左下角的"开始"按钮（Windows图标）

2. 在搜索框输入：cmd

3. 看到黑色的"命令提示符"图标

4. 点击它打开
```

**方法C：右键开始菜单**
```
1. 右键点击左下角的"开始"按钮

2. 选择"命令提示符"或"Windows PowerShell"

3. 窗口就打开了
```

---

### 步骤2：进入项目目录

**如果项目在 C:\Users\YourName\Downloads\industry-risk-advisor**

```
在黑色的命令窗口中输入：

cd C:\Users\YourName\Downloads\industry-risk-advisor

然后按回车键 Enter

提示：
- cd 是"change directory"的缩写，意思是"切换目录"
- 您需要把路径改成您实际的路径
```

**不知道项目在哪里？**
```
方法1：查看下载位置
- 如果是从GitHub下载的，查看浏览器的下载记录

方法2：用搜索功能
- 按 Win 键，输入文件夹名称
- 右键点击搜索结果 → 打开文件位置
```

---

### 步骤3：运行命令

**现在，在命令窗口中输入：**

```
python src/main.py
```

**然后按回车键 Enter**

**看到类似这样的输出就成功了：**
```
INFO:     Started server process
INFO:     Waiting for application startup.
INFO:     Application startup complete.
INFO:     Uvicorn running on http://0.0.0.0:5000
```

## 📍 方法2：Mac系统

**方法A：用Spotlight搜索**
```
1. 按 Command + 空格键（同时按）

2. 输入：terminal

3. 按回车键

4. 终端窗口就打开了！
```

```
1. 打开"访达"（Finder）

3. 找到"实用工具"文件夹
```

```
然后按回车键 Enter
```

```
python src/main.py
```

**按回车键 Enter**

## ❓ 常见问题

### Q1: 提示"python不是内部或外部命令"

**原因：**
```
您的电脑没有安装Python，或者没有配置环境变量
```

**解决方法：**
```
1. 访问 https://www.python.org/downloads/
2. 下载 Python 3.12
3. 安装时，一定要勾选：
   ✅ Add Python to PATH
4. 安装完成后，重新打开命令窗口
5. 再次运行命令
```

---

### Q2: 提示"找不到路径"

**原因：**
```
您不在项目目录中
```

**解决方法：**
```
先运行：
cd 项目路径

例如：
cd C:\Users\YourName\Downloads\industry-risk-advisor

然后再运行：
python src/main.py
```

---

### Q3: 提示"ModuleNotFoundError"

**原因：**
```
缺少依赖包
```

**解决方法：**
```
先安装依赖：

pip install -r requirements.txt

然后再运行：
python src/main.py
```

---

### Q4: 我不知道项目文件夹在哪里

**解决方法：**
```
方法1：查看下载位置
- 如果是从GitHub下载的
- 打开浏览器，查看下载记录
- 右键点击下载的文件 → 打开所在文件夹

方法2：用搜索功能
- Windows：按 Win 键，输入文件夹名称
- Mac：按 Command + 空格，输入文件夹名称

方法3：问AI
- 告诉我您的操作系统
- 告诉我您从哪里下载的项目
- 我帮您找到路径
```
