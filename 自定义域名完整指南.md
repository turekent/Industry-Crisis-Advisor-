# 🌐 自定义域名完整指南

---

## 一、域名能否解决您的问题？

### ✅ 答案：可以！

**使用自定义域名的好处：**

```
✅ 解决微信拦截问题
   └─ 微信对自定义域名更友好

✅ 更专业
   └─ yourdomain.com 比 xxx.vercel.app 更专业

✅ 更稳定
   └─ 不受 Vercel 默认域名限制

✅ 可以绑定多个项目
   └─ 一个域名可以创建多个子域名
```

---

## 二、一个域名能否绑定多个项目？

### ✅ 答案：可以！一个域名可以绑定很多项目！

**原理：使用子域名**

```
主域名：yourdomain.com

可以创建无限个子域名：
├── industry.yourdomain.com    → 产业避险参谋
├── pcb.yourdomain.com         → PCB专项分析
├── yacht.yourdomain.com       → 游艇行业分析
├── news.yourdomain.com        → 新闻系统
└── app.yourdomain.com         → 其他应用

每个子域名可以绑定一个 Vercel 项目！
```

---

### 步骤2：在 Vercel 绑定域名（5分钟）

```
1. 登录 Vercel
2. 进入您的项目
3. 点击 Settings → Domains
4. 输入您的域名，如：industry.yourdomain.com
5. 点击 Add
```

---

### 步骤3：配置 DNS 解析（5分钟）

**在域名服务商添加记录：**

#### 配置示例（以阿里云为例）：

```
记录类型：CNAME
主机记录：industry
记录值：cname.vercel-dns.com
TTL：10分钟
```

**添加第二条记录：**
```
记录类型：A
主机记录：industry
记录值：76.76.21.21
TTL：10分钟
```

---

### 步骤4：等待生效

```
• DNS 解析生效：1-24 小时
• 通常 5-10 分钟即可
• 生效后即可使用自定义域名访问
```

## 五、多项目绑定示例

### 场景：您有多个项目

**假设您购买了：`riskadvisor.com`**

#### 配置方案：

```
项目1：产业避险参谋
──────────────────────────
域名：industry.riskadvisor.com
DNS 记录：
  CNAME industry → cname.vercel-dns.com
  

项目2：PCB 专项分析
──────────────────────────
域名：pcb.riskadvisor.com
DNS 记录：
  CNAME pcb → cname.vercel-dns.com
  

项目3：游艇行业分析
──────────────────────────
域名：yacht.riskadvisor.com
DNS 记录：
  CNAME yacht → cname.vercel-dns.com
  

项目4：主网站
──────────────────────────
域名：riskadvisor.com 或 www.riskadvisor.com
DNS 记录：
  A @ → 76.76.21.21
  CNAME www → cname.vercel-dns.com
```

## 七、常见问题

### Q1: 域名购买后多久能用？

```
国内平台（阿里云/腾讯云）：
  • 需要实名认证：1-3 天
  • DNS 解析生效：1-24 小时
  • 总计：2-4 天

国际平台（Namecheap）：
  • 无需认证，立即可用
  • DNS 解析：5-10 分钟
  • 总计：10 分钟
```

### Q2: 一个域名最多能绑定多少项目？

```
理论上：无限个

实际限制：
  • DNS 记录数：通常无限制
  • Vercel 项目：免费版可绑定多个域名
  
建议：
  • 一个项目用一个子域名
  • 主域名（如 yourdomain.com）用于主项目
```

### Q3: 子域名和主域名有什么区别？

```
主域名：
  yourdomain.com
  → 用于主网站

子域名：
  industry.yourdomain.com
  pcb.yourdomain.com
  yacht.yourdomain.com
  → 用于不同的项目/功能
```

### Q4: 域名需要备案吗？

```
如果服务器在国内：
  ✅ 必须备案（需要 20-30 天）

如果服务器在国外（Vercel 在国外）：
  ❌ 不需要备案
  
结论：
  Vercel 部署 → 不需要备案！
```
