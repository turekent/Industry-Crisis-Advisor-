# 🛡️ 产业避险参谋

基于AI的战争动态成本预警系统

## 功能

- 📰 实时市场动态（彭博社、路透社等权威来源）
- 💰 核心价格指标（原油、铜、铝、钯金、黄金、SCFI运费）
- 🔍 行业风险查询（PCB、游艇、生物医药等六大行业）
- 📊 成本影响分析
- 💡 决策建议

## 技术栈

- 后端: Python (FastAPI, LangChain, LangGraph)
- 前端: HTML/CSS/JavaScript
- AI: 豆包大模型

## 快速开始

```bash
pip install -r requirements.txt
python src/main.py
